# Everyday Finance · Personal Budget Assistant

A Python Tkinter‑based graphical tool for managing personal finances. It supports multiple accounts, transaction logging, category analysis, monthly forecasting, and CSV/JSON import/export.

## Project Overview

This project is the implementation of the “Personal Budget and Spending Assistant” topic for COMP1110. It provides a simple graphical interface to help users track daily income and expenses, and offers insights such as category breakdowns, monthly projections, and subscription spending.

## Programming Language & Environment

- **Language**: Python 3.10+
- **Dependencies**: Uses only Python standard libraries (`tkinter`, `csv`, `json`, `datetime`, `pathlib`, etc.). No third‑party packages are required.

## How to Run

1. Ensure Python 3.10 or later is installed on your system.
2. Open a terminal (or command prompt) and navigate to the project folder.
3. Run the following command:
   ```bash
   python main.py
